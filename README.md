# FR4GE

A set of experimental tools for managing the lifecycle of printed circuit board assemblies. The name FR4GE is a word play on "[FR-4](https://en.wikipedia.org/wiki/FR-4)" and is pronounced like "[forge](https://en.wikipedia.org/wiki/Forge)" (/fôrj/).
